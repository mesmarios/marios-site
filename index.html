Boosting Loop — Αλγοριθμική Ροή
Κάθε επανάληψη προσθέτει ένα δέντρο που διορθώνει τα σφάλματα του προηγούμενου
Μαθηματικό Υπόβαθρο
obj(t) = Σ l( yi, ŷi(t) )  +  Σ Ω( fi )

· MSE για regression
· Log-loss για classification
· Custom objectives
② Regularization Ω
Ελέγχει την πολυπλοκότητα του μοντέλου.

Αποτρέπει overfitting:
· T = αριθμός φύλλων
· w = βάρη φύλλων
· γ, λ = hyperparameters
Regularization — Αποφεύγοντας το Overfitting
Ω(f) = γT  +  ½λ Σ wj²
γ (gamma) Minimum loss reduction
για split. Μεγαλύτερο = απλούστερο δέντρο.
T Αριθμός φύλλων
στο δέντρο
λ (lambda) L2 regularization
στα βάρη φύλλων
wj Βάρος (score)
κάθε φύλλου j
— Good fit   ╌ Overfitting
Χωρίς Regularization → Overfitting
Με Regularization → Γενίκευση
Taylor Expansion — Γιατί Είναι Γρήγορος
L(t) ≈ Σ [gi · ft(xi)  +  ½hi · ft²(xi)]  +  Ω(ft)
gi
Gradient
1η παράγωγος loss
hi
Hessian
2η παράγωγος loss
Η χρήση 2ης παραγώγου = ακριβέστερη βελτιστοποίηση από τo απλό Gradient Boosting
Optimal leaf weight:  w*j = −(Σgi) / (Σhi + λ)

from xgboost import XGBClassifier
from sklearn.metrics import accuracy_score

# Train model
model = XGBClassifier(
    n_estimators=100,
    learning_rate=0.1,
    max_depth=6
)
model.fit(X_train, y_train)

# Evaluate
predictions = model.predict(X_test)
print(f"Accuracy: {accuracy_score(y_test, predictions):.2%}")
